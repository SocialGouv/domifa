Consulter le guide SMS

Information importante : afin que les domiciliés reçoivent des SMS, veuillez compléter les 3 étapes ci-dessous :

  1. Activez les SMS via le formulaire ci-dessous
  2. Complétez le nom de l'expéditeur et la signature des SMS. Seules des lettres peuvent être utilisées pour l'expéditeur, aucun numéro ne doit figurer dans l'expéditeur !
  3. Dans la section dossier des fiches domiciliés, ajoutez un numéro de portable à chaque domicilié

Vous pouvez consulter un document explicatif sur la mise en place des notifications SMS.
